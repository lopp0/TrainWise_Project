## Architecture cheat-sheet

- Backend: `Controllers → BL → DAL → DBservice` (no EF, no migrations folder; SPs live in SSMS).
- Frontend: navigation in `src/navigation/NavigationStack.js`; canonical axios client is `src/services/api.js` (base URL `http://192.168.1.117:5249`); legacy `src/api/api.js` is being phased out but still hosts `getActivityLogs` + `registerUser`.
- Auth: session-based, no JWT. `useAuth()` exposes `userId` from AsyncStorage.
- Health Connect: read-only sync, requires manual edits in `MainActivity.kt` + `AndroidManifest.xml`. **Never** run `npx expo prebuild --clean` — it overwrites those edits.
- Time: backend stores UTC ISO strings as-sent; display layers convert to `Asia/Jerusalem`.
